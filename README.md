> *somewhere between a coder and a curator,*  
> *between logic and lullaby,*  
> *between syntax and silence —*  
> *there lives **laiba**.*

i build small, careful things on the internet.  
i believe interfaces should feel like **morning light** —  
present without demanding attention.

i write code the way some people write postcards:  
**slowly, kindly, and meant for someone in particular.**
