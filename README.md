# Artistic Van Gogh Starry Night Theme

## Description
This project features a mesmerizing artistic theme inspired by Van Gogh's "Starry Night." It incorporates real-time dynamic elements and a cosmic aesthetic, including swirling animations and starry backgrounds.

## Features
- **Starry Background**: A background that simulates the beautiful night sky, filled with shimmering stars and cosmic elements.
- **Swirling Animations**: Elements of the UI that smoothly transition and swirl, echoing Van Gogh’s painting techniques.
- **Dynamic Content**: Real-time updates that bring life to the artwork, making it feel as if the stars are twinkling and the sky is in motion.

## Installation
To install this theme, simply clone the repository and open the index.html file in your web browser.
